## 下载

### Docker

```
docker pull apachecn0/team-wiki
docker run -tid -p <port>:80 apachecn0/team-wiki
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install team-wiki
team-wiki <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g team-wiki
team-wiki <port>
# 访问 http://localhost:{port} 查看文档
```
